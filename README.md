# 🍽️ Zomato Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing Zomato restaurant data using Python. The goal is to extract meaningful insights from the dataset such as restaurant trends, ratings, pricing, and customer preferences.

The project was created as part of a learning exercise by following a YouTube tutorial from the channel **The iScale**.

---

## 🎯 Objectives
- Understand the structure of real-world datasets
- Perform data cleaning and preprocessing
- Analyze restaurant data to uncover patterns
- Visualize insights using graphs and charts

---

## 📂 Dataset
The dataset used in this project contains information about:
- Restaurant names
- Locations
- Ratings
- Cost for two
- Online delivery availability
- Table booking availability
- Votes and reviews

---

## 🛠️ Technologies Used
- **Python**
- **Pandas** – Data manipulation and analysis
- **NumPy** – Numerical operations
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical data visualization
- **Jupyter Notebook**

---

## 🔍 Key Analysis Performed
- Data cleaning and handling missing values
- Distribution of ratings
- Cost analysis of restaurants
- Online delivery vs ratings comparison
- Table booking impact on ratings
- Location-based restaurant analysis

---

## 📊 Visualizations
The project includes various visualizations such as:
- Bar charts
- Count plots
- Heatmaps
- Distribution plots

These help in better understanding the patterns and trends in the dataset.
